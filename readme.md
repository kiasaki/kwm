# kwm

a minimalistic x11 window manager.

- hover to focus
- click to raise
- super+move to move
- super+shift+move to resize
- super+tab to cycle
- super+q to quit
- super+m to maximise a window
- super+t to launch terminal
- super+w to launch browser
- super+space to launch dmenu
- super+f1 to kill the wm
